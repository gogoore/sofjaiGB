# 项目简介

## 核心功能
- 系统首页管理
- 个人中心管理
- 业主信息管理
- 物业经理信息管理
- 维修专员信息管理
- 综合客服信息管理
- 系统管理
- 房产信息管理
- 资产信息管理
- 报修信息管理
- 维修记录管理
- 投诉信息管理
- 缴费信息管理
- 财务报表管理

## 适用场景
- 小区物业管理
- 业主日常信息查询与操作
- 物业经理工作流程管理
- 维修服务跟踪与记录
- 综合客服服务支持

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
